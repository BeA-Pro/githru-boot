# 실행 방법
.env 파일 생성 후, 다음과 같이 사용자 token에 token 값을 넣어주세요.
```
GITHUB_AUTH_TOKEN='사용자 token'
```
terminal
```
npm install
npm test
```
terminal을 통해 결과 값을 확인할 수 있습니다.
